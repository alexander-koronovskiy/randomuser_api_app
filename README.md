Описание:

Приложение для взаимодействия с внешним API: https://randomuser.me/
Основная страница приложения представляет из себя простую web страницу c полем для ввода количества человек, которые необходимо загрузить с API, а также информация о случайно выбранном человеке.


Функционал:

1) На страницу выводится следующая информация: пол человека, имя, фамилия, номер телефона, email, место проживания.

2) При запросе от клиента - событии при нажатии "UPDATE USER" каждый раз выдаётся случайный человек из базы данных.

3) Поле для ввода количества человек (находится в шапке web-страницы) загружает в таблицу информацию о пользователях из базы данных. Количество строк в таблице определяется введенным с клавиатуры числом от 0 до 10.

4) Каждый раз при старте приложения - его база данных обновляется.

5) При нажатии "LOAD DATA" происходит скачивание json-файла с информацией, содержащейся в базе данных.


Техническая информация:

1) Используются фреймворки для работы с бд и клиентом: Flask, Sql lite, peewee.

2) Когда сервер стартует, он загружает 10 случайных людей из API.

3) Выполнение программы происходит внутри docker контейнера.


Сборка программы:

1) Создание образа контейнера: $ docker build -t my_app .
2) Запуск контейнера из собранного образа: $ docker run -p 5000:5000 -t my_app
3) Переход по адресу в браузере: /localhost:5000


Тестирование:

По умолчанию должен быть установлен python

1) $ pip install -r requirements.txt
2) Переход к содержимому, в котором находятся тесты: $ cd app
3) $ pytest


В разработке:

Обращение к пользователю по адресу: http://homepage/user_id.
