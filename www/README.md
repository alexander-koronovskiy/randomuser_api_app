Описание:

Приложение для взаимодействия с внешним API: https://randomuser.me/
Основная страница приложения представляет из себя простую web страницу
c полем для ввода количества человек, которые необходимо загрузить с API,
а также таблица с информацией о случайно выбранном человеке.

Функционал:

Храниться следующая информация: пол человека, имя, фамилия, номер телефона, email, место проживания.
При перезагрузке страницы таблица с информацией обновляется случайным образом.
Вся информация о людях хранится в базе данных.
При запросе от клиента каждый раз выдаётся случайный человек из базы данных.

Техническая информация:

Используется Flask, Sql lite, pewee. Когда сервер стартует, он загружает 10 случайных людей из API.
Интерфейс имеет кнопку, по нажатию на которую происходит обновление информации
в таблице без перезагрузки исходной страницы, загрузка инфо из БД в виде JSON файла.
Выполнение программы происходит внутри docker контейнера.


Сборка программы с использованием docker:

1) Создание образа контейнера: $ docker build -t my_app .
2) Запуск контейнера из собранного образа: $ docker run -p 5000:5000 -t my_app
3) Переход по адресу /localhost:5000

Сборка без использвания docker:

1) Clone /homework_final from https://github.com/alexander-koronovskiy/epam_homeworks.git
2) Run app/index.py
3) Переход по адресу /localhost:5000

В разработке:

Покрытие тестами. Обращение к пользователю по адресу: http://homepage/user_id.
