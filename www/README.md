Описание:

Приложение для взаимодействия с внешним API: https://randomuser.me/
Основная страница приложения представляет из себя простую web страницу c полем для ввода количества человек, которые необходимо загрузить с API, а также информация о случайно выбранном человеке.


Функционал:

На страницу выводится следующая информация: пол человека, имя, фамилия, номер телефона, email, место проживания (1).
При перезагрузке страницы - информация о пользователе обновляется случайным образом (2), при запросе от клиента - событие при нажатии "UPDATE USER" каждый раз выдаётся случайный человек из базы данных (1, 2).

(1)						(2)


Техническая информация:

Используется Flask, Sql lite, pewee. Когда сервер стартует, он загружает 10 случайных людей из API.
Интерфейс имеет кнопку, по нажатию на которую происходит обновление информации
в таблице без перезагрузки исходной страницы, загрузка инфо из БД в виде JSON файла.
Выполнение программы происходит внутри docker контейнера.


Сборка программы:

1) Переход в папку приложения: $ cd /www
2) Создание образа контейнера: $ docker build -t my_app .
3) Запуск контейнера из собранного образа: $ docker run -p 5000:5000 -t my_app
4) Переход по адресу в браузере: /localhost:5000

Тестирование:

1) Переход к приложению с тестами: $ cd /www/app
2) $ pytest


В разработке:

Обращение к пользователю по адресу: http://homepage/user_id.
